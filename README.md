# Trios — Modern Websites for Local Businesses

A premium web agency website built with React 18 + Vite 5 + Tailwind CSS 3.

**Live at:** [https://trios-web.vercel.app](https://trios-web.vercel.app)

## Tech Stack

| Tool | Version | Purpose |
|------|---------|---------|
| React | 18.3 | UI framework |
| Vite | 5.4 | Build tool |
| Tailwind CSS | 3.4 | Styling |
| React Router | 6.23 | Client-side routing |
| Lucide React | latest | Icons |
| EmailJS | latest | Contact form |
